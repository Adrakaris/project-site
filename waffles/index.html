WAFFLES

Overview

WAFFLES, Warwick's Amazing Fast Food Logistic Engagement Servie, is a coursework for CS126: Data Structures and Algorithms. The premise of Waffles is that you are responsible for developing a back-end system for a restaurant review site, which contains restaurants, customers, favourites and reviews of those customers. They should be added, and able to be sorted and served up in a reasonable about of time.

When data is loaded in, there is a good deal of bad or invalid data, thus we needed to catch, deal with, and alter data, all in a timely manner.

The most important thing to all of this is having good data structures and efficient algorithms, which we had to implement manually as part of the coursework, hash maps and binary trees being some of the most essential data structures to ensure efficiency of algorithms.
